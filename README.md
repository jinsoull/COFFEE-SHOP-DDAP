# COFFEE-SHOP-DDAP
# ☕ Brew Haven Coffee

## Deskripsi Project

Brew Haven Coffee merupakan website landing page untuk coffee shop yang dirancang dengan menerapkan prinsip User Interface (UI) dan User Experience (UX). Website ini bertujuan untuk memberikan informasi mengenai menu, profil coffee shop, galeri, ulasan pelanggan, serta layanan reservasi meja.

Website dibuat sebagai tugas individu mata kuliah Desain Antarmuka Pengguna dengan fokus pada desain yang modern, navigasi yang jelas, tata letak yang rapi, serta pengalaman pengguna yang nyaman.

## Target Pengguna

## Penerapan Prinsip UI/UX

### Konsistensi

Penggunaan warna, tipografi, tombol, dan card diterapkan secara konsisten pada seluruh halaman.

### Hierarki Visual

Ukuran heading, warna, dan tata letak digunakan untuk membantu pengguna memahami informasi dengan mudah.

### Feedback

Tombol dan card memiliki efek hover sebagai umpan balik visual terhadap interaksi pengguna.

### Navigasi

Navigation bar dibuat sederhana dan mudah digunakan untuk berpindah antar section.

### Layout

Layout menggunakan grid dan whitespace yang cukup sehingga tampilan lebih rapi dan nyaman dibaca.
